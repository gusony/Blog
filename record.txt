SSH quick connect: 
1.in ~/.ssh/config
2.vim ~/.ssh/config
    Host short-name
    HostName domain-name
    User username
    Port 22
3.in terminal :SSH short-name
4.connected

SSH keygen:
1.ssh-keygen -t rsa (generate rsa key to ~/.ssh/id_rsa.pub)
2.1:ssh-copy-id -i .ssh/id_rsa.pub hostname # 推薦V
2.2:scp id_rsa.pub server_hostname:~/.ssh # (擇一)缺點, 機器沒有.ssh目錄時，會有問題
3:finish
